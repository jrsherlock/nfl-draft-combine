— Built on MiroFish · tuned for the draft

The general-purpose swarm engine, pointed at the room.

MiroFish is a multi-agent simulation engine that builds a parallel digital world from real-world signals, populates it with persona-driven agents, and lets the future rehearse itself. DRO is the version that knows the NFL draft.

Persona generation  →  525 named decision-makers — GMs, area scouts, position coaches, analytics directors, owners, beat writers, certified agents — each with a scheme bias, recency curve, risk tolerance, and a comp-set memory mined from a decade of that team's actual picks.

GraphRAG knowledge graph  →  every prospect's tape, medical, character flags, scheme fit, agency, contract leverage; every team's needs, cap room, draft capital, prior trade partners, ownership patience.

Reality seed extraction  →  continuous ingest of combine, pro days, top-30 visits, beat-writer leaks, mocks, market lines.

Parallel simulation  →  ten thousand full first rounds run side-by-side — each one a different sequence of leaks, debates, owner overrides, and trade-ups.

Dynamic temporal memory  →  boards mutate as picks land. When CHI takes Mason at one, twenty other rooms re-rank. DRO watches the cascade.

Counterfactual sandbox  →  queue dozens of "what if" perturbations and run them in parallel. What if QB1 fails his medical? What if a top-5 GM is fired draft week? Each dish surfaces the trends, hidden risks, and second-order reactions that teams running a single timeline would never see.

Deep interaction  →  ask any simulated GM why he passed. Ask the swarm what flips Walker into the top-five. The model talks back.

A draft board doesn't form. It's negotiated. DRO models the negotiation.
